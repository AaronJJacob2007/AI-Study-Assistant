# AI Semester Companion

An AI-powered academic assistant built using Python, Streamlit, and the Gemini API.

The project was created to learn AI application development, API integration, and software engineering concepts while building a practical tool for students. It helps users learn concepts, generate quizzes, create study notes, and build personalized study plans.

## Features

### AI Tutor

* Ask questions on academic topics
* Receive detailed AI-generated explanations
* Difficulty-based responses (Beginner, Intermediate, Advanced)

### Quiz Generator

* Generate multiple-choice quizzes on any topic
* Subject-specific quiz generation
* Difficulty-level customization

### Notes Generator

* Create structured study notes
* Includes definitions, key concepts, advantages, disadvantages, applications, and revision summaries

### Study Planner

* Generate study plans based on a specific topic
* Generate study plans for an entire subject
* Customizable based on the number of days until the exam

### User Experience Features

* Subject selection
* Difficulty selection
* Input validation
* Loading indicators during AI processing
* Error handling for API failures
